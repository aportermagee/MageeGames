# JavaScript Games

## Snake
- A simple js game where the player controls a snake that moves around and eats food boxes
- If the snake crashes into a wall or itself, the game ends
- Everytime the snake eats a food box the score increases by one and the snake gains a segment

## Tetris 
- Simple js version of classic tetris game
- The player tries to fill rows to eliminate them by moving and rotating four-pieced blocks
- If a block reaches the top, the game ends

## Space Invaders
- Simple js version of the original space invaders game
- The player controls a small ship at the bottom of the screen and attempts to shoot all of the enemy ships which are shooting back
- The player has three lives, when the player is hit with one life left, the game ends
- If an enemy reaches the red line, the game ends
- Bullets collide with each other and with the green defenses that protect the player
